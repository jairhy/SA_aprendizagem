Fundamentos de Eletroeletrônica

1. Componentes Eletrônicos Empregados em Serviços

Resistor

O resistor é um componente passivo que oferece resistência elétrica ao fluxo de corrente, limitando a quantidade de corrente que passa por um circuito. É usado para controlar tensão e corrente, dividir tensão e proteger componentes sensíveis.

Tipos: Fixo, variável (potenciômetro), dependente (NTC, PTC)

Unidade: Ohm (Ω)

Aplicações: Divisores de tensão, limitadores de corrente, pull-up/down

Diodo LED (Light Emitting Diode)

O diodo LED é um semicondutor que emite luz quando polarizado diretamente. Converte energia elétrica em luz de forma eficiente, com longa vida útil e baixo consumo.

Características: Baixo consumo, longa duração, cores variadas

Tensão típica: 1.8V a 3.3V (depende da cor)

Aplicações: Indicadores, displays, iluminação, sinalização

Transistor

O transistor é um componente semicondutor usado como amplificador ou chave. Controla corrente maior com sinal menor, sendo fundamental para circuitos digitais e analógicos.

Tipos: BJT (Bipolar Junction Transistor), MOSFET, JFET

Funções: Amplificação, comutação, modulação

Aplicações: Amplificadores, chaves digitais, reguladores de tensão

Circuito Integrado (CI)

O circuito integrado é um conjunto de componentes eletrônicos miniaturizados em um único chip de silício. Pode conter milhares ou milhões de transistores e outros componentes.

Tipos: Analógicos, digitais, mistos

Exemplos: Microcontroladores (Arduino), processadores, memórias

Aplicações: Computadores, smartphones, sistemas embarcados

2. Sensores Empregados em Sistemas Automatizados

Sensor de Temperatura

Mede a temperatura ambiente ou de superfícies, convertendo variação térmica em sinal elétrico.

Tipos: NTC (coeficiente negativo), PTC, termopar, DS18B20

Faixa: -55°C a +150°C (depende do tipo)

Aplicações: Controle climático, refrigeração, monitoramento industrial

Sensor de Umidade

Mede a umidade relativa do ar, detectando a quantidade de vapor d'água presente.

Tipos: Capacitivo, resistivo, DHT11, DHT22

Faixa: 0% a 100% UR

Aplicações: Controle de umidade, estações meteorológicas, agricultura

Sensor de Luminosidade

Detecta a intensidade de luz ambiente, usando fotocélulas ou LDRs (resistores dependentes de luz).

Tipos: LDR, fototransistor, fotodiodo

Faixa: 0 a 100.000 lux (aproximadamente)

Aplicações: Controle automático de luzes, câmeras, painéis solares

Outros Sensores Comuns

Sensor de Movimento (PIR): Detecta presença através de infravermelho

Sensor de Som: Microfone para detecção de ruído

Sensor de Distância: Ultrassônico ou infravermelho para medição

Sensor de Pressão: BMP180 para pressão atmosférica

Sensor de Gás: MQ-135 para detecção de gases tóxicos

3. Arduino: O que é, Para que Serve / Aplicações, Vantagens e Desvantagens

O que é Arduino?

O Arduino é uma plataforma de prototipagem eletrônica open-source baseada em hardware e software fácil de usar. Consiste em uma placa com microcontrolador e um ambiente de desenvolvimento integrado (IDE) para programar o dispositivo.

Para que Serve?

O Arduino serve para criar projetos interativos que podem sentir o ambiente através de sensores e controlar dispositivos como motores, LEDs e relés. É ideal para prototipagem rápida de ideias em eletrônica e automação.

Aplicações

Domótica: Automação residencial (luzes, cortinas)

Robótica: Controle de robôs e veículos

IoT: Dispositivos conectados à internet

Educação: Ensino de eletrônica e programação

Arte: Instalações interativas e iluminação

Indústria: Controle de processos e monitoramento

Vantagens

Fácil de usar: Interface amigável e comunidade ativa

Open-source: Hardware e software gratuitos

Versátil: Múltiplas placas para diferentes necessidades

Econômico: Baixo custo de entrada

Extensível: Shields e módulos para expandir funcionalidades

Desvantagens

Performance limitada: Não substitui computadores industriais

Alimentação: Dependente de fonte externa ou bateria

Precisão: Relógio interno menos preciso que cristais externos

Memória: Limitada para aplicações complexas

4. Como o Arduino Funciona? O que é Eletrônica Embarcada?

Como o Arduino Funciona

O Arduino funciona através de um microcontrolador (como o ATmega328) que executa programas armazenados na memória flash. O programa lê entradas (sensores), processa dados e controla saídas (atuadores). A comunicação com o computador ocorre via USB para programação e depuração.

Ciclo de Funcionamento

1. Alimentação: Energia fornecida via USB, bateria ou fonte externa

2. Inicialização: Setup do programa (configuração de pinos)

3. Loop principal: Execução contínua do código em loop()

4. Leitura: Sensores fornecem dados analógicos/digitais

5. Processamento: Microcontrolador executa lógica programada

6. Saída: Controle de LEDs, motores, displays

O que é Eletrônica Embarcada?

Eletrônica embarcada é a integração de componentes eletrônicos em sistemas dedicados para realizar funções específicas. Inclui microcontroladores, sensores e atuadores em um único dispositivo, otimizado para tarefas particulares com baixo consumo e tamanho reduzido.

Características da Eletrônica Embarcada

Especializada: Projetada para função específica

Integrada: Hardware e software em um sistema

Eficiente: Baixo consumo de energia

Autônoma: Opera sem intervenção humana constante

Conectada: Pode se comunicar com outros sistemas

5. Arduino: Qual a Linguagem de Programação Usada

Linguagem do Arduino

O Arduino utiliza uma variante da linguagem C++ adaptada para microcontroladores. Não usa Python diretamente, embora existam implementações alternativas como MicroPython para algumas placas.

Por que C++?

Eficiência: Código compilado roda rápido em hardware limitado

Controle: Acesso direto ao hardware (registradores, pinos)

Compatibilidade: Baseado em AVR-GCC, padrão da indústria

Comunidade: Ampla documentação e exemplos disponíveis

Estrutura Básica do Programa Arduino

void setup() {
    // Configuração inicial - executada uma vez
    pinMode(13, OUTPUT);
}

void loop() {
    // Loop principal - executado continuamente
    digitalWrite(13, HIGH);
    delay(1000);
    digitalWrite(13, LOW);
    delay(1000);
}

Alternativas

MicroPython: Para placas como ESP32, mais simples que C++

Arduino Language: Simplificação do C++ com funções prontas

Outros: Assembly para otimização extrema, mas raro

6. Entradas e Saídas Digitais no Arduino

O que são Entradas Digitais?

Entradas digitais no Arduino lêem apenas dois estados: HIGH (5V ou 3.3V) ou LOW (0V). Usadas para botões, interruptores, sensores de presença, etc.

Como Configurar Entrada Digital

pinMode(2, INPUT);  // Pino 2 como entrada
int estado = digitalRead(2);  // Lê estado: HIGH ou LOW

O que são Saídas Digitais?

Saídas digitais produzem tensão HIGH ou LOW para controlar dispositivos como LEDs, relés, buzzers, etc. Podem fornecer corrente limitada (cerca de 20mA por pino).

Como Configurar Saída Digital

pinMode(13, OUTPUT);  // Pino 13 como saída
digitalWrite(13, HIGH);  // Liga saída (5V)
digitalWrite(13, LOW);   // Desliga saída (0V)

Exemplo Prático: Botão Controlando LED

const int botao = 2;
const int led = 13;

void setup() {
    pinMode(botao, INPUT);
    pinMode(led, OUTPUT);
}

void loop() {
    if (digitalRead(botao) == HIGH) {
        digitalWrite(led, HIGH);
    } else {
        digitalWrite(led, LOW);
    }
}

7. Entradas e Saídas Analógicas no Arduino

O que são Entradas Analógicas?

Entradas analógicas convertem sinais contínuos (0-5V) em valores digitais (0-1023). Usadas para sensores que produzem tensão variável, como potenciômetros, sensores de temperatura, etc.

Como Ler Entrada Analógica

int valor = analogRead(A0);  // Lê pino A0 (0-1023)
float tensao = valor * (5.0 / 1023.0);  // Converte para volts

O que são Saídas Analógicas (PWM)?

O Arduino não tem saídas analógicas verdadeiras, mas simula com PWM (Pulse Width Modulation). Produz sinal quadrado com duty cycle variável, simulando tensão média entre 0-5V.

Como Usar Saída PWM

analogWrite(9, 128);  // Pino 9 com 50% duty cycle (2.5V médio)
analogWrite(9, 255);  // 100% duty cycle (5V)
analogWrite(9, 0);    // 0% duty cycle (0V)

Exemplo: Controle de Brilho de LED

const int pot = A0;
const int led = 9;

void setup() {
    pinMode(led, OUTPUT);
}

void loop() {
    int valor = analogRead(pot);
    int brilho = map(valor, 0, 1023, 0, 255);
    analogWrite(led, brilho);
}

8. Script para Cálculo da Tensão, Corrente, Resistência pela Lei de Ohm

Lei de Ohm

A Lei de Ohm estabelece que a tensão (V) é igual à corrente (I) multiplicada pela resistência (R): V = I × R

Fórmulas Derivadas

Corrente: I = V / R

Resistência: R = V / I

Tensão: V = I × R

Script JavaScript para os Cálculos

function calcularOhm() {
    let v = parseFloat(document.getElementById('tensao').value);
    let i = parseFloat(document.getElementById('corrente').value);
    let r = parseFloat(document.getElementById('resistencia').value);

    if (v && i && !r) {
        r = v / i;
        document.getElementById('resistencia').value = r.toFixed(2);
    } else if (v && !i && r) {
        i = v / r;
        document.getElementById('corrente').value = i.toFixed(3);
    } else if (!v && i && r) {
        v = i * r;
        document.getElementById('tensao').value = v.toFixed(2);
    }

    document.getElementById('ohmResult').style.display = 'block';
}

9. Script para Cálculo da Potência

Fórmulas de Potência Elétrica

Potência = Tensão × Corrente: P = V × I

Potência = Corrente² × Resistência: P = I² × R

Potência = Tensão² / Resistência: P = V² / R

Calculadora de Potência

Potência calculada!

Script JavaScript para Cálculo de Potência

function calcularPotencia() {
    let v = parseFloat(document.getElementById('potTensao').value);
    let i = parseFloat(document.getElementById('potCorrente').value);
    let r = parseFloat(document.getElementById('potResistencia').value);

    let p = 0;

    if (v && i) {
        p = v * i;
        alert(`Potência = ${p.toFixed(2)} W (V × I)`);
    } else if (i && r) {
        p = i * i * r;
        alert(`Potência = ${p.toFixed(2)} W (I² × R)`);
    } else if (v && r) {
        p = (v * v) / r;
        alert(`Potência = ${p.toFixed(2)} W (V² / R)`);
    } else {
        alert('Forneça pelo menos dois valores para calcular a potência');
        return;
    }

    document.getElementById('potenciaResult').style.display = 'block';
}

10. Programa para Leitura de Sensor Analógico Acionando Saídas Digitais

Exemplo: Sensor de Luminosidade Controlando LED

Este programa lê um sensor de luminosidade (LDR) conectado ao pino analógico A0. Se a luminosidade estiver baixa, aciona um LED no pino digital 13.

Circuito Sugerido

LDR: Pino A0 ↔ Resistor 10kΩ ↔ GND

LED: Pino 13 ↔ Resistor 220Ω ↔ GND

Código Arduino

// Programa: Leitura sensor analógico controlando saída digital
const int sensorLuz = A0;    // Pino analógico do sensor
const int led = 13;         // Pino digital do LED
const int limite = 500;     // Limite para acionar LED

void setup() {
    pinMode(led, OUTPUT);      // Configura LED como saída
    Serial.begin(9600);        // Inicia comunicação serial
}

void loop() {
    int valorSensor = analogRead(sensorLuz);  // Lê sensor (0-1023)

    Serial.print("Luminosidade: ");
    Serial.println(valorSensor);  // Exibe valor no monitor serial

    if (valorSensor < limite) {  // Se luminosidade baixa
        digitalWrite(led, HIGH);  // Liga LED
        Serial.println("LED LIGADO - Ambiente escuro");
    } else {
        digitalWrite(led, LOW);   // Desliga LED
        Serial.println("LED DESLIGADO - Ambiente claro");
    }

    delay(1000);  // Aguarda 1 segundo
}

Funcionamento

Leitura Analógica: Sensor fornece valor de 0 (escuro) a 1023 (claro)

Comparação: Se valor < 500, considera ambiente escuro

Saída Digital: LED acende quando luminosidade baixa

Monitor Serial: Exibe valores para depuração

11. O que é um Diodo? Um Diodo LED? Um Display 7 Segmentos?

O que é um Diodo?

O diodo é um componente semicondutor que permite a passagem de corrente elétrica em apenas uma direção. É formado pela junção de dois materiais semicondutores dopados (PN).

Funcionamento Básico

Polarização Direta: Corrente flui (ânodo positivo, cátodo negativo)

Polarização Reversa: Corrente não flui (bloqueia tensão)

Tensão de Limiar: ~0.7V para diodos de silício

Tipos de Diodos

Diodo Retificador: Converte AC para DC

Diodo Zener: Regula tensão (polarização reversa)

Diodo Schottky: Baixa tensão de limiar

Diodo LED: Emite luz (ver abaixo)

O que é um Diodo LED?

O diodo LED (Light Emitting Diode) é um tipo especial de diodo que emite luz quando polarizado diretamente. A luz é produzida pela recombinação de elétrons e buracos na junção PN.

Características do LED

Cores: Vermelho, verde, azul, branco, etc.

Tensão: 1.8V-3.3V (depende da cor)

Corrente: 10-20mA típica

Vida Útil: 50.000-100.000 horas

Resistor: Sempre usar resistor em série para limitar corrente

O que é um Display 7 Segmentos?

O display 7 segmentos é um componente formado por 7 LEDs (e às vezes 1 ponto decimal) arranjados em formato de "8". Cada segmento pode ser ligado individualmente para formar dígitos de 0 a 9 e algumas letras.

Funcionamento do Display 7 Segmentos

Segmentos: a, b, c, d, e, f, g (e dp para ponto decimal)

Tipo: Ânodo comum ou cátodo comum

Controle: 7-8 pinos digitais do Arduino

Aplicações: Relógios digitais, contadores, displays numéricos

Exemplo: Controlando Display 7 Segmentos

// Display 7 segmentos - Cátodo comum
const int segmentos[7] = {2, 3, 4, 5, 6, 7, 8}; // Pinos a-g

// Padrões para dígitos 0-9 (LOW = ligado em cátodo comum)
const byte digitos[10] = {
    B11000000, // 0
    B11111001, // 1
    B10100100, // 2
    B10110000, // 3
    B10011001, // 4
    B10010010, // 5
    B10000010, // 6
    B11111000, // 7
    B10000000, // 8
    B10010000  // 9
};

void setup() {
    for(int i = 0; i < 7; i++) {
        pinMode(segmentos[i], OUTPUT);
    }
}

void loop() {
    for(int digito = 0; digito < 10; digito++) {
        mostrarDigito(digito);
        delay(1000);
    }
}

void mostrarDigito(int digito) {
    for(int i = 0; i < 7; i++) {
        digitalWrite(segmentos[i], bitRead(digitos[digito], i));
    }
}
